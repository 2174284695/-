# 路线模拟器 v1.7.0

本版本聚焦路线规划可用性和地图查看体验：移除驾车吸附入口，改为更明确的步行道路
吸附与规划；同时把底部面板改为可收起，并把往返路线改成一键勾选的原路返回模式。

## 路线规划

- 移除“驾车吸附”功能入口，避免在步行模拟场景中产生混淆。
- “道路吸附”现在固定按步行道路处理手绘路线。
- 已有路线可继续手绘完善，支持从终点继续画，也支持从起点前补画。
- “多点规划”不再弹出步行/驾车选择，点击后直接按步行道路规划。
- 如果已经点选了途经点，会按点选顺序依次经过这些点。
- 新增道路点 / 强制经过点模型：多点规划会自动判断途经点是否明显远离道路。
- 自动途经点距离规划道路约 10 米以上时，会转为橙色强制经过点，并把原始坐标插回
  最终路线，避免非道路区域途经点被路由服务忽略。
- 靠近道路线段的途经点会保持为蓝色道路点，避免因为道路几何点稀疏而误判。
- 点选多个途经点后，可以在“点选”模式下拖动途经点调整通过顺序。
- 支持长按途经点打开编辑菜单，可切换道路点 / 强制经过点，或删除单个途经点。
- 途经点现在会显示顺序编号；蓝色表示道路点，橙色表示强制经过点。
- 强制经过点相邻的离路线段会以橙色虚线显示。
- 新增“顺序编辑”，可在弹窗中精确上移或下移途经点。
- 新增“定位设起点”，可把手机真实当前位置插入为路线或途经点起点。
- 新增“反转顺序”，可一键对调路线起点和终点方向。
- 如果没有途经点，但屏幕上已有路线，会直接使用当前路线作为规划输入。
- 规划失败时会用红线标出本次失败的大概规划范围。
- 规划失败弹窗新增“重新规划 / 保留原路线 / 撤销规划”三选项。

## 速度模拟

- 默认加入起步加速和到达减速。
- 转弯路段会自动减速。
- 长直线路段会略微加速。
- 经过强制经过点相邻的离路线段时会进一步降速，并带短距离过渡。
- 上述变化会和原有自然速度波动叠加，并继续受到加速度限制。

## 往返路线

- “往返路线”改为勾选框。
- 勾选后，模拟会先从起点到终点，再按原路线从终点返回起点。
- 往返不会改写原始路线，也不会把返程点永久保存进路线库。
- 路线距离、预计时间和预计步数会按往返总距离显示。

## 地图查看

- 底部面板新增路线状态提示条，会显示当前状态和下一步建议。
- 主界面重组为“编辑方式 / 路线模式 / 设定速度 / 主动作 / 更多工具”。
- “开始模拟 / 多点规划 / 路线库 / 更多工具”成为已有路线时的主动作区。
- 空路线时优先显示定位设起点、点选、手绘和路线库入口。
- 更多工具按“绘制与规划 / 路线管理 / 模拟高级”分组展开。
- 自然速度波动和 Health Connect 移入“模拟高级”。
- 底部操作面板新增“收起 / 展开”按钮。
- 底部操作面板限制最大高度，功能较多时可在面板内滚动。
- 按钮高度、间距和面板留白进一步压缩，减少对地图的遮挡。
- 收起后只保留一行标题和展开按钮，减少对地图、定位按钮和缩放按钮的遮挡。
- 地图右侧控件仍会跟随底部面板高度自动调整位置。

## 使用建议

1. 手绘路线后点击“道路吸附”，生成步行道路路线。
2. 点选多个途经点后点击“多点规划”，按点选顺序规划步行路线。
3. 从路线库载入路线后，也可以直接点击“多点规划”，把当前路线重新贴合到道路。
4. 需要到达终点后按原路返回时勾选“往返路线”，再开始模拟。

## 说明

道路吸附与多点规划依赖联网路由服务。若网络不可用、公共服务限流，或当地道路数据
缺失，应用会保留原路线并显示失败原因。

当前编辑状态会保存途经点类型；新保存到路线库的命名路线也会保存道路点 / 强制经过点
元数据，载入后会恢复蓝色/橙色途经点。旧版本已经保存的路线没有这部分元数据，载入后
仍可使用路线几何，再次规划时会根据坐标重新自动判断。
