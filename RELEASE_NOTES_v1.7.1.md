# 路线模拟器 v1.7.1

这是 v1.7 系列的发布补丁版，用于把最新的混合路线编辑、强制途经点和路线模拟体验升级正式同步到 GitHub Release。

## 更新内容

- 新增道路点 / 强制经过点模型，解决非道路区域途经点被多点规划忽略的问题。
- 多点规划会保留强制经过点，并用橙色虚线标记离路线段。
- 支持长按途经点切换道路点 / 强制经过点，或删除单个途经点。
- 支持途经点编号、拖拽排序、顺序编辑、反转顺序和定位设起点。
- 已有路线可以继续从起点或终点手绘补全。
- 勾选“往返路线”后，会先到终点再按原路线返回起点。
- 模拟速度默认加入起步加速、到达减速、转弯减速和直线略加速。
- 底部操作面板支持收起 / 展开，减少对地图查看和定位按钮的遮挡。

## 说明

道路吸附和多点规划依赖联网路由服务。若网络不可用、公共服务限流，或当地道路数据缺失，应用会保留原路线并显示失败原因。

第三方应用可以识别并拒绝 Android 模拟位置。本项目不会绕过此类检测，仅用于 Android 系统允许的模拟位置开发、测试和路线验证场景。
