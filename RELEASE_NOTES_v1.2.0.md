# 路线模拟器 v1.2.0

首个公开发布版本，面向 Android 开发、导航原型和定位场景测试。

## 主要功能

- 在 MapLibre 地图上点选或手绘路线
- 路线命名保存、载入和删除
- 一键定位到手机当前真实位置
- 以 1.0～30.0 km/h 的设定速度沿路线移动
- 支持 ±1%～±15% 的自然速度波动
- 平滑随机速度漂移和加速度限制
- 实时估算模拟步数
- Android 14+ 可选 Health Connect 步数同步
- 前台服务持续运行，可从通知栏停止

## v1.2.0 更新

- 新增路线库，可保存多条命名路线
- 新增路线载入、覆盖更新和删除功能
- 新增地图右侧“一键定位”按钮
- 修复应用退出到后台再返回后地图显示异常的问题
- 地图样式不可用时会在回到前台后自动尝试恢复

## 安装

1. 下载并安装下方 APK。
2. 在 Android“开发者选项”中打开“选择模拟位置信息应用”。
3. 选择“路线模拟器”。
4. 开启系统定位服务并授予精确位置权限。

系统要求：Android 8.0（API 26）或更高版本。

## 已知限制

- 路线沿绘制折线移动，不会自动吸附到道路。
- 普通应用无法向硬件 `TYPE_STEP_COUNTER` 注入事件；不读取 Health Connect 的第三方
  计步应用可能不会显示模拟步数。
- 第三方应用可以识别并拒绝 Android 模拟位置，本项目不会绕过此类检测。
- 默认地图依赖网络连接和 OpenFreeMap 服务。

Android 官方测试位置会保留模拟位置属性。使用标准 Location API 且未主动禁止模拟
位置的应用通常可以收到坐标，但目标应用仍可根据模拟标记、设备完整性和自身风控策略
拒绝数据，因此无法保证兼容所有第三方软件。
